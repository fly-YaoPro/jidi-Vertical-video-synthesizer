# 更新日志

## v1.5

### 视觉与设计

- 将旧版偏 Windows 原生控件的界面升级为 Liquid Glass 风格。
- 统一采用半透明玻璃质感、柔和高光、圆角卡片、轻量阴影和蓝白渐变背景。
- 优化整体比例，让界面从“大而空”调整为更小巧、精致的竖版工具布局。
- 统一窗口外框和内部控件圆角，减少不协调的直角痕迹。
- 优化拖动窗口时 UI 跟随慢半拍的问题，减少大面积模糊带来的卡顿感。

### 窗口与标题栏

- 自定义顶部标题栏，改为类 macOS 红绿灯按钮样式。
- 鼠标悬停窗口按钮时显示对应操作提示图形。
- 修复顶部标题没有真正居中的问题，现在标题按整个窗口居中显示。
- 标题栏结构调整为更接近 macOS app 的顶部布局。

### 拖拽与文件选择

- 横屏视频和封套 PNG 保持同一行方块布局，减少横向空白。
- 上传成功后卡片会变为选中状态，并显示绿色状态反馈。
- 修复上传后 icon 显示成绿色小方块的问题，改为更清晰的矢量图标样式。
- 优化拖拽卡片的视觉反馈，让选择文件的状态更直观。
- 选择横屏视频后，默认输出目录自动设置为视频所在目录。

### 控件与交互

- 重新整理模板 / 自定义切换控件，恢复横向排列，并加入更自然的滑动反馈。
- 修复下拉列表默认展开、横向排列、无法收起、层级遮挡等问题。
- 修复部分下拉菜单透明、模糊失效、选项无法点击的问题。
- 统一下拉框、输入框、导出设置、高级参数等控件的圆角、背景和玻璃质感。
- 点击开始生成后不再自动跳转到日志页，保持在操作界面。

### 布局与自适应

- 修复缩小窗口后内容被横向遮挡的问题。
- 优化窗口尺寸变化时的自适应表现，减少右侧内容被裁切。
- 调整底部“开始生成”和进度条区域，确保窗口缩小时仍然可见。
- 后续发布只保留单文件 exe，不再单独发布快速启动版。

### 导出能力

- 新增导出质量三档：最佳 `15 Mbps`、高 `8 Mbps`、普通 `6 Mbps`。
- 默认导出质量调整为最佳 `15 Mbps`。
- 自动编码模式下显示预计使用方式，例如 `GPU / NVENC` 或 `CPU x264`。
- 自动检测 NVENC，自动模式下优先使用 GPU，不支持时回退 CPU。

### 完成反馈和彩蛋

- 生成完成后加入从底部扫过全界面的高光动画。
- 将传统弹窗改为背景模糊后的完成浮层。
- 增加绿色圆形对勾动画，参考 Apple Pay 成功反馈的感觉。
- 完成界面增加“打开文件所在位置”和“OK”按钮。
- 修复 OK 按钮文字颜色看不清的问题。
- 输出完成后显示居中的小气泡文案：
  - 工作时间 09:00-20:00：`项目结束，今天你又是Be Different的一天。`
  - 加班时间 20:01-次日 08:59：`加班辛苦了！今天的你Committed爆了！`
- 修复气泡文字没有居中、气泡整体不在 UI 横向中心的问题。

### 性能

- 优化启动流程，不再打开软件时阻塞等待 NVENC 检测。
- 软件先快速打开界面，再后台检测编码能力。
- 减少大面积玻璃模糊强度，提升拖动窗口和界面响应速度。
