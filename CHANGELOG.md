# 更新日志

## v1.5

### 新增优化

1. 界面焕新，加入丝滑动效和光感交互。
2. 优化整体比例，界面更小巧、精致，竖版工具布局。
3. 拖入视频后，自动匹配该视频所在目录为输出目录。
4. 整合为单 exe 文件，即点即用。
5. 优化代码层，提高加载和导出速度。
6. 新增【双彩蛋】，期待你的发现！

### 解决 Bug

1. 修复顶部标题没有真正居中的问题，现在标题按整个窗口居中显示。
2. 统一窗口外框和内部控件圆角，减少不协调的直角痕迹。
3. 修复上传后 icon 显示成绿色小方块的问题，改为更清晰的矢量图标样式。
4. 修复下拉列表默认展开、横向排列、无法收起、层级遮挡等问题。
5. 修复部分下拉菜单透明、模糊失效、选项无法点击的问题。
6. 优化窗口尺寸变化时的自适应表现，减少右侧内容被裁切。
7. 优化拖动窗口时 UI 跟随慢半拍的问题，减少大面积模糊带来的卡顿感。
8. 单文件 exe 启动速度已优化，后续发布只保留单文件版本。
