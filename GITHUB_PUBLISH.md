```markdown
## 基地边缘竖版视频合成器Pro v1.3

轻量 Windows 桌面工具，用于将横屏视频自动合成为固定封套样式的 1080x1260 竖版 MP4。

### 功能

- 拖入横屏视频和封套 PNG
- 自动合成 1080x1260 竖版视频
- 支持 NVIDIA NVENC / CPU x264 编码
- 支持最佳、高、普通三档导出质量
- 支持模板命名和自定义命名
- 支持完成后定位输出文件

### 使用方法

下载 zip 后解压，保持以下文件在同一目录：

- 基地边缘竖版视频合成器Pro_v1.3.exe
- ffmpeg.exe
- ffprobe.exe

双击 exe 即可运行。
```
